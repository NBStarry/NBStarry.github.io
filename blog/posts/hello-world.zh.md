# Hello World — 欢迎来到我的博客

这是我全新的个人主页和博客，从零开始用原生 HTML、CSS 和 JavaScript 构建。

## 为什么重新开始？

我决定告别之前基于 Hexo 的博客，自己动手搭建一个全新的网站。自己理解网站上的每一行代码，没有黑魔法、没有隐藏的抽象层，只有纯粹的 Web 基础技术——这种感觉很棒。

## 接下来会写什么

我计划分享以下方面的内容：

- **游戏开发** — 游戏设计思考、Unity 技巧、交互体验
- **AI & 深度学习** — 从图神经网络到生成式 AI 的最新动态
- **生物学与计算** — 探索生物信息学与计算生物学
- **随想杂谈** — 生活、工具推荐，以及一切有趣的事

## 技术栈

这个网站使用了：

- 纯 **HTML5**、**CSS3** 和 **原生 JavaScript**
- 基于 Canvas 的粒子动画星空背景
- GitHub API 集成，实时展示项目数据
- **marked.js** 实现客户端 Markdown 渲染
- 部署在 **GitHub Pages** 上，零构建步骤
