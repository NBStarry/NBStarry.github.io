# Hello World — Welcome to My Blog

This is my new personal homepage and blog, rebuilt from the ground up with vanilla HTML, CSS, and JavaScript.

## Why a Fresh Start?

I decided to move away from my previous Hexo-based blog and build something custom. There's something satisfying about understanding every line of code on your own website — no magic, no hidden abstractions, just pure web fundamentals.

## What to Expect

I plan to write about:

- **Game Development** — Thoughts on game design, Unity tips, and interactive experiences
- **AI & Deep Learning** — From graph neural networks to the latest in generative AI
- **Biology meets Computing** — Exploring bioinformatics and computational biology
- **Random Musings** — Life at SJTU, tools I love, and everything in between

## The Tech Stack

This site is built with:

- Pure **HTML5**, **CSS3**, and **vanilla JavaScript**
- Canvas-based particle animation for the starfield background
- GitHub API integration for live project data
- **marked.js** for client-side Markdown rendering
- Deployed on **GitHub Pages** with zero build step
